App and device security analysis

BLEmap analyses Bluetooth Low Energy (BLE) devices and the applications that interact with them. BLEmap combines these two analysis to extract device functionality and identify security vulnerabilities present in IoT and other BLE-enabled devices.

Vulnerabiilty and device database

BLEmap holds functionality and security data from thousands of devices. Our application analysis algorithm allows us to identify the main functionality of a device and its security. Our database can also be used to match devices in the real world, allowing to identify vulnerable devices by scanning the area for Bluetooth devices.

Automatic analysis

BLEmap is automatic and can be used from a command line (to enable automation), or with an easy-to-use graphical user interface. BLEmap will generate reports in machine and human readable formats. BLEmap will soon be available in a mobile application, making it even easier to perform Bluetooth security analysis on-the-go..
